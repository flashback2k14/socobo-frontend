# socobo-project

## Steps:
---------
1. install gulp with **npm install -g gulp**
2. run **npm install**
3. run **bower install**
4. start application with **gulp serve**

## Documentation:
-----------------
http://localhost:63342/socobo-frontend/app/elements/documentation.html

## Testing:
-----------
